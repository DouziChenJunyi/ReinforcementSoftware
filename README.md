# ReinforcementSoftware_Qt
 这是“面向XX的XX计算机平台加固软件”。该项目基于Qt框架进行开发，包含五大模块：CPU加固、内存加固、寄存器加固、总线加固、I/O加固等，每个模块包含文件选择、进程调用、文本渲染等功能。  由于部分核心代码不便公布，此项目只开源部分代码。
